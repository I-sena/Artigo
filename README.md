# Análise de Dados: ARTIGO-CSBC

Este repositório contém os scripts, dados e documentos utilizados para o desenvolvimento do artigo submetido ao Congresso da Sociedade Brasileira de Computação (CSBC). O projeto envolve análise exploratória de dados (EDA) utilizando Python e R, focando na autoavaliação institucional.

## 📂 Estrutura do Repositório

- Documentos Principais:

  - `Proposta_instrumento_de_autoavaliacao.pdf`: Proposta de instrumento de autoavaliação institucional.
  - `Proposta_curso.pdf`: Proposta de instrumento de avaliação de curso.
  - `Proposta_docentes.pdf`: Proposta de instrumento de avaliação de docentes por discentes.

- [`Graficos-Python`](Graficos-Python): Contém scripts em Python para geração de visualizações específicas.

  - `imagens/`: Pasta onde os gráficos gerados são salvos.

  - `boxplot_eixo2.py` / `boxplotCampi.py`: Scripts para geração de boxplots.

  - `radarEixos.py`: Script para criação de gráficos de radar.

  - `satisfacao.py`: Script para criação de gráficos com análises de níveis de satisfação.

  - `*.txt`: Arquivos de dados por campus (Arraias, Gurupi, Miracema, Palmas, Porto Nacional).

- [Script-R](Script-R): Contém a análise estatística principal.

  - `analiseDados.rmd`: Arquivo R Markdown com a análise detalhada e geração de relatórios.

## 🛠️ Tecnologias Utilizadas

Python 3.12 | Bibliotecas do projeto:  `pandas`, `matplotlib`, `seaborn`, `pyplot`, `numpy`.

R / RStudio | Dependências do projeto: `tidyverse`, `scales`, `corrplot`, `ggrepel`, `fmsb`, `kableExtra`, `knitr`, `dplyr` e `tidyr`.

LaTeX: Para a redação do artigo acadêmico.
